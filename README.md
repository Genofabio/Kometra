# KomaLab

**KomaLab** è un software scientifico per il trattamento e l'analisi di immagini astronomiche di comete, sviluppato come progetto di tesi per la Laurea Magistrale in Ingegneria Informatica presso il Politecnico di Torino.

Questo strumento è creato in stretta collaborazione con gli astronomi dell'Osservatorio Astrofisico di Asiago (INAF) ed è progettato per lo studio e l'analisi delle proprietà fisiche e chimiche delle comete attraverso l'elaborazione avanzata delle immagini.
